# Contributing to grok-terminal-mcp

Thank you for considering contributing!

## Development Setup

```bash
cd tools/grok-terminal-mcp
npm install
npm run dev
```

## Code Style

- Use TypeScript
- Keep the security model explicit and auditable
- Prefer clarity over cleverness
- Add tests for security-sensitive or complex logic

## Pull Requests

- Describe the problem you're solving
- Keep changes focused
- Update documentation if behavior or configuration changes

## Security

If you find a security issue, please report it privately first (see SECURITY.md if present, or contact the maintainer directly).
